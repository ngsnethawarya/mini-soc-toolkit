# Mini SOC Toolkit

This repository contains beginner-friendly Python scripts for basic security operations practice.

## Included tools
- Log reader
- Failed login counter
- IP address extractor
- Password strength checker
- Hash generator
- File integrity checker
- Phishing keyword scanner
- CSV alert summary

## Purpose
This project shows how Python can support basic cybersecurity tasks in a Security Operations Center environment.

## Skills practiced
- Python automation
- Log analysis
- Indicator extraction
- Alert review
- Security awareness
- Incident response basics

## Who this helps
This repo is useful for beginners learning cybersecurity, students building a portfolio, and anyone practicing defensive security concepts.
